# Psychic Game

#### Project Description

Psychic Game randomly selects a letter and then the user has to guess which letter it is within 9 tries. 

Deployed site: https://mattomjoe.github.io/Psychic-Game/


#### Key Features

* List of guesses tried so far

* Wins and losses
